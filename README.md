# Ask Map

**Ask a question, get an interactive map.**

Ask Map is an AI-powered web app that turns any topic into an interactive map
with pins, routes, knowledge cards, and more.

🌐 [askmap.app](https://askmap.app)

![Welcome](assets/4.jpg)

## Features

**Knowledge Cards** — Each place gets a card with historical context, scene images, and character portraits.

![Spanish Armada](assets/1.jpg)

**3D Globe & Arc Routes** — Explore on a 3D globe with great-circle arc arrows and distance labels.

![Magellan](assets/2.jpg)

**Character Bio Cards** — AI-generated biographies with portraits on a 3D terrain map.

![Crusade](assets/3.jpg)

**Multilingual** — Works in English, Chinese, Japanese, and more.

![Pearl Harbor](assets/5.jpg)
